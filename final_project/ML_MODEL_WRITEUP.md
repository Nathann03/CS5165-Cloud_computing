# ML Model Write-Up

## Linear Regression
Linear Regression is a supervised learning method used to predict a continuous numeric value from one or more input variables. It assumes that the relationship between predictors and the target can be approximated as linear, which makes the model fast, interpretable, and easy to explain in a business setting. In retail analytics, it can be used as a baseline for predicting household spend, basket value, or customer lifetime value. The main strength of Linear Regression is clarity: stakeholders can understand the direction and approximate size of each variable’s effect. Its main weakness is that it does not naturally capture non-linear behavior, interactions, and diminishing returns, all of which appear frequently in real shopping data. For this project, Linear Regression is best viewed as a baseline model rather than the final production choice for CLV.

## Random Forest
Random Forest is an ensemble model that combines many decision trees and aggregates their predictions. Each tree is trained on a random sample of the data and a random subset of features, which helps reduce overfitting and improves generalization. Random Forest works well for both regression and classification and is especially useful when the data contains non-linear patterns or important interactions among variables. In retail analytics, it is a strong option for churn classification, promotion response prediction, and basket-level modeling because it can learn relationships among demographics, recency, basket composition, and spending behavior without requiring heavy preprocessing. Its strengths are solid performance, robustness, and flexibility. Its main tradeoff is that it is less interpretable than a simple linear model. In this project, Random Forest is used for basket analysis because it supports a clear cross-sell question and performs well on structured retail basket features.

## Gradient Boosting
Gradient Boosting is an ensemble technique that builds a sequence of weak learners, usually shallow decision trees, where each new tree is trained to correct the errors made by earlier trees. This stage-by-stage improvement process often produces strong predictive accuracy on structured tabular business data. In retail analytics, Gradient Boosting is useful for customer lifetime value and churn modeling because it captures non-linear effects, interactions, and threshold behavior that simpler models often miss. Its advantages are high predictive power and strong performance on well-engineered features. Its tradeoffs are longer training time and lower interpretability than Linear Regression. For this project, Gradient Boosting is the main CLV method and also supports churn risk modeling because it balances accuracy with manageable implementation complexity in a coursework setting.
