Azure Deployment Load Path

For Azure deployment, the preferred path is to store the raw CSV files in Azure Blob Storage and run the separate loader script so the web app does not perform heavy ingestion during startup.

  • python3 scripts/upload_dataset_to_blob.py uploads the provided CSV files to Blob Storage.
  • python3 scripts/load_blob_to_db.py downloads those blobs and populates the managed Azure database.
  • The web app should then start in read-only mode against the already-populated database.
