ML Model Application

Basket Analysis With A Clear Retail Question

Retail question: What product combinations are commonly purchased together, and how can they drive cross-selling opportunities?

The app answers that with a Random Forest model trained at basket level. It predicts whether a likely cross-sell target commodity will appear when a basket already contains a high-frequency primary commodity.

Output shown below: selected commodities, model accuracy, ROC AUC, and saved JSON metrics for submission.
