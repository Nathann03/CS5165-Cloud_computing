# Submission Helper

## What To Turn In
- Azure web app URL
- Repository or zip of the project code
- `ML_MODEL_WRITEUP.md`
- `TEST_REPORT.md`
- `FINAL_STATUS.md`
- Key screenshots listed below

## Recommended Screenshots
- Home page showing the username/password/email form
- Sample data pull page for `HSHD_NUM 10`
- Search page with a successful household query
- Dashboard page with charts visible
- Admin reload page with the load report
- ML page showing CLV, basket, and churn metrics
- Azure Portal view showing the Web App and PostgreSQL Flexible Server

## Demo Flow
1. Open the home page and point out the rubric-required form.
2. Open the sample pull for household 10.
3. Use the search page to query another household.
4. Show the dashboard and explain one spending trend plus one basket pair.
5. Open the ML page and summarize CLV, basket, and churn outputs.
6. Show the reload page and explain that fresh CSVs can be re-imported.
7. Show the Azure resource list and the live app URL.
